# Analysis of N-Queens Solver

## Problem description

The N-Queens problem is a classic puzzle that asks how to place N chess queens on an N x N chessboard so that no two queens threaten each other. In other words, no two queens can share the same row, column, or diagonal.

## Implementation details

The code provided implements a backtracking algorithm to solve the N-Queens problem. The `solve` function takes a `board` vector that represents the current state of the board and an integer `row` that represents the row that is currently being checked. The function iterates over all possible positions in the row and recursively calls itself for each position that is valid (i.e., does not conflict with any previously placed queens). When a solution is found (i.e., `row == board.size()`), the `print_board` function is called to print the solution to the console.

The `is_valid` function is used to check whether a given position conflicts with any previously placed queens. It checks whether the column, the diagonal from top-left to bottom-right, or the diagonal from bottom-left to top-right is already occupied.

The code also includes functionality to run the solver multiple times and compute statistics on the execution time and the number of solutions found.
